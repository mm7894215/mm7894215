### About

Indie developer focused on **local-first** software — tools that run on your machine, keep your data yours, and feel native on every platform.

I care about the small things: clean typography, restrained color, and interfaces that get out of the way.

<br/>

### Featured · TokenTracker

A local-first AI token usage tracker. One dashboard for every AI CLI you run.

- **Cross-platform** — CLI · macOS menu-bar app · Windows tray app · web
- **Private by design** — token counts only, never your prompts or messages
- **Multi-provider** — Claude, Codex, Cursor, Gemini, and more

```
AI CLI runs  →  hook fires  →  parse  →  local dashboard
```
